XML是一种可扩展标记语言，被设计用来传输和存储数据，比如将结果返回给客户端。HTML是呈现数据的。HTML中都是预定义标签（提前已经有标准，定义好的），而XML中没有预定义标签，全是自己定义的。

之前Ajax在进行数据交换的时候所使用的数据格式就是XML，现在使用的格式是JSON。

Ajax的优点：

  • 无需刷新页面就可以与服务器通信（搜索时的提示词）

  • 允许根据用户的事件来更新部分页面的内容（淘宝中用户将鼠标放在侧面的表单上便会出现整个分类栏）

缺点：

  • 没有浏览历史，不能回退
  • 存在跨域问题
  • Ajax向服务端发送请求，服务端返回结果，用js动态创建的页面，无法用爬虫来获取数据的

全局刷新和局部刷新

  1. 全局刷新

    • 用表单form、herf等发起的请求是全局刷新，用户发起请求，视图发生了改变，页面发生跳转。

    • 缺点：渲染新的页面需要时间，占用了网络带宽。如果数据量很大，那么耗时将很长，用户体验性不好。

  2. 局部刷新

    • 在当前页面中发送请求，获取数据，更新当前页面的dom对象，对视图部分刷新。

    • 优点：①只获取少量数据，速度传输更快。

      ​ ②部分更新页面，速度很快。

      ​ ③在一个页面中，可以做多个局部刷新

异步请求对象

在局部刷新中，使用异步请求对象，在浏览器的内部发送请求来获取数据。

异步对象是在浏览器内存中的JavaScript对象。

可以创建多个，相互独立。每个对象都可以独立的发起请求，传递参数，接收数据。效率更高。

异步对象

创建方法：
var xhr = new XMLHttpRequest();

使用异步对象主要负责发起请求，传递请求的参数，并从服务器接收数据

局部刷新需要使用的技术：

  1. JavaScript：创建异步对象，调用它的属性和方法
  2. dom：处理dom来更新数据
  3. css：处理视图，美化
  4. servlet：服务器端技术
  5. json

把以上这些技术综合应用就叫做Ajax，他不是新的编程语言，而是多种技术的综合应用。AJax全称为asynchronous JavaScript and XML（异步的JavaScript和XML）。我们使用Ajax来实现局部刷新
